# NCE Brand Palette

## Colors

| Name         | Hex Code  | Usage                        |
|--------------|-----------|------------------------------|
| White        | #ffffff   | Backgrounds, text on dark    |
| Black        | #000000   | Text, borders                |
| Citron       | #D7DF23   | Primary accent, CTAs, active states |
| Charcoal     | #231F20   | Primary dark, headers, text  |
| Graphite     | #373536   | Secondary dark, hover states |
| Silver       | #E7E7E6   | Light backgrounds, borders   |
| Medium Gray  | #D0CFCD   | Borders, dividers            |
| Smoke        | #141618   | Darkest, overlays            |

## CSS Variables

Use these in custom CSS:

```css
var(--nce-white)
var(--nce-black)
var(--nce-citron)
var(--nce-charcoal)
var(--nce-graphite)
var(--nce-silver)
var(--nce-medium-gray)
var(--nce-smoke)

/* Semantic */
var(--nce-primary)
var(--nce-secondary)
var(--nce-background)
var(--nce-text)
var(--nce-border)
var(--nce-font-family)
```
